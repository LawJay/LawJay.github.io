Honours Project 2018


Hemiplegic User Experience

Due to the rapid advancement of technology in the consumer market, there is a niche area of users whose needs are not being met by current systems. Assistive technology is an umbrella term that describes assistive, adaptive and rehabilitative devices for people with disabilities. Whilst the majority of video game hardware is designed for able-bodied users, the same cannot be said for Hemiplegic users. (The term Hemiplegia relates to the paralysis or weakness of one side of the body) There are however some software applications that assist these individuals but have limitations and do not offer the full user experience. The aim of this project is to explore this area of technology and attempt to create part of a solution, capable of managing more intricate computing tasks. Research conducted during the project revealed that the only way to create a solution for the end user was to give them the tool to create their own solution. Through the usage of speech command technology, Utilizing the Microsoft Speech recognition engine and the C# .Net Framework, a Windows Presentation Foundation (WPF) application was created to allow users with zero background in software development or computer science, to create “Modals” a file containing a Speech command activator phrase, and a key to execute once the command had been spoken. This allows users of all physical capabilities to add an extra form of input to their PC, enabling them to activate certain keyboard commands whilst physically typing on different keys on the keyboard, therefore enabling hemiplegic users to perform overlapping button sequences that they could not otherwise do. Although this software greatly benefits those affected by Hemiplegia, the software could also help users of sound upper body limb usage.
